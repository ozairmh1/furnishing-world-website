# Furnishing World — Premium Interior Business Website

A modern and responsive single-page business website developed for an interior and furnishing business. The project focuses on delivering a premium user experience with smooth animations, elegant UI design, mobile responsiveness, and interactive frontend components.

## ✨ Features

* Modern luxury-inspired UI/UX
* Fully responsive design
* Smooth scrolling and animations
* Interactive navigation menu
* Mobile-friendly layout
* Animated hero section
* Gallery showcase section
* Contact and business information section
* Optimized frontend structure

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Tailwind CSS
* Google Fonts

## 📱 Responsive Design

The website is optimized for:

* Desktop
* Tablet
* Mobile devices

## 🚀 Project Purpose

This project was created as a real-world business landing page solution for showcasing services, business information, and customer contact details in a clean and professional format.

## 🎨 UI Highlights

* Premium furnishing/interior theme
* Custom animations and transitions
* Elegant typography
* Modern section layouts
* Interactive gallery effects
* Smooth user experience
